# es6-gas-template

google app scriptをes6で書く際のテンプレートです。

## ES6からGASファイルに変換

`browserify -t babelify -p gasify [変換対象のファイルへのパス] -o [変換後のコードを書き込むファイルへのパス]`

## GASのダッシュボードへpush

`clasp push`
